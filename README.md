# Atom Icons

A collection of file icons from various glyph fonts.  Based off of [Seti-UI](https://github.com/jesseweed/seti-ui) and [file-icons](https://github.com/DanBrooker/file-icons).

## Fonts
Contains fonts from the following
* [FontMfizz](http://mfizz.com/oss/font-mfizz)(`.mf`)
* [Icomoon](http://icomoon.io)(`.iconmoon`)
* [Octicons](https://octicons.github.com/)(`.octicons`)

### Installation

```
cd ~/.atom/packages
git clone https://github.com/unstablereality/atom-icons --depth=1
```
